El elemento <div> permite agrupar un conjunto de elementos juntos en un cuadro de nivel de bloque. Por ejemplo, puede crear un elemento <div> para que contenga todos los elements del encabezado del sitio ( el logotipo y la navegacion) o puede crear un elemento <div> para que contenga informacion en bloques.

en un navegador, el contenido del elemento <div> comenzara en una nueva linea, pero aparte de eso no ara ninguna diferencia en la presentacion de la pagina. sin embargo, el uso de un atributo ID o class en el elemento <div> significa que puede crear reglas de estilo CSSpara indicar cuanto espacio debe ocupar el elemento <div> en la pantalla y cambiar la apariencia de todos los elementos contenido en el Tambien puede facilitar el seguimiento del codigo si ha utilizado elementos <div> para contener cada seccion de la pagina.

dado que puede haber varios otros elementos dentro de un elemento <div> puede ser utilr agregar un comentario despues de la etiqueta </div> de cierre. Esto le permite ver claramente a que etiqueta de apertura se supone corresponde.
